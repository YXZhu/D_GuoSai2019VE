# D_GuoSai2019VE
## 简易电路特性测试仪 19国赛
## Make By 国赛走一波
* 使用ARM9内核的全志F1C100工控屏做图像界面
* 使用TP2272放大器作为信号的缓冲和放大
* 使用STM32F407的DAC产生200k正弦波作为测试仪信号输出
* 使用STM32F407的ADC作为特性电路的信号采集
* 分析放大器电路的输入电阻、输出电阻、频谱特性曲线以及其上限频率
